# Preparation

```bash
$ pip install -r requirement.txt
$ npm install -g grunt
$ npm install
$ bower install
```

### Build

```bash
$ grunt coffee stylus
```

or if you want to change source of JavaScript or CSS

Activate watching task and edit them as you want.

```bash
$ grunt
```


# Starting

```bash
$ python application.py
```
